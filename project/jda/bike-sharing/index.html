About this Project

Bike Sharing Project

Untuk proyek ini, saya menggunakan Python untuk mengimpor dataset yang berekstensi .csv, melakukan cleaning/pembersihan, menganalisa dan memvisualisasikan dataset dari Kaggle bernama Bike Sharing Dataset. Bike Sharing System adalah sebuah generasi baru dari rental sepeda tradisional, dimana seluruh proses dari keanggotaan, rental dan pengembalian kembali menjadi otomatis. Melalui sistem ini pengguna bisa dengan mudah menyewa sepeda dari posisi tertentu ke posisi lainnya.

Dataset

Pada proyek ini hanya terdapat 2 buah table yaitu day.csv dan hour csv yang berisi tentang tanggal, musim, tahun dan bulan penyewaan sepeda serta informasi lainnya.

Kedua tabel memiliki kolom yang sama kecuali hr pada tabel hour.csv yang tidak tersedia pada tabel day.csv.

Berikut keterangan mengenai kedua tabel:

  • instant: indeks catatan
  • dteday : tanggal
  • season : musim (1: musim semi, 2: musim panas, 3: musim gugur, 4: musim dingin)
  • yr : tahun (0: 2011, 1: 2012)
  • mnth : bulan (1 sampai 12)
  • hr : jam (0 hingga 23)
  • holiday : hari cuaca hari libur atau tidak (diambil dari [Tautan Web])
  • weekday : hari dalam seminggu
  • workingday : jika hari tersebut bukan akhir pekan atau hari libur maka nilainya 1, jika tidak maka nilainya 0.
  • weathersite :
    • Cerah, Sedikit awan, Berawan sebagian, Berawan sebagian
    • Kabut + Berawan, Kabut + Awan pecah, Kabut + Sedikit awan, Kabut
    • Salju Ringan, Hujan Ringan + Badai Petir + Awan berserakan, Hujan Ringan + Awan berserakan
    • Hujan Lebat + Butiran Es + Badai Petir + Kabut, Salju + Kabut
  • temp: Suhu yang dinormalisasi dalam Celcius. Nilai diperoleh melalui (t-t_min)/(t_max-t_min), t_min = -8, t_max = +39 (hanya dalam skala per jam)
  • atemp: Suhu perasaan yang dinormalisasi dalam Celcius. Nilai-nilai ini diperoleh melalui (t-t_min)/(t_max-t_min), t_min=-16, t_max=+50 (hanya dalam skala per jam)
  • hum: Kelembapan yang dinormalisasi. Nilai dibagi menjadi 100 (maksimum)
  • windspeed: Kecepatan angin yang dinormalisasi. Nilai dibagi menjadi 67 (maks)
  • casual: jumlah pengguna biasa
  • registered: jumlah pengguna terdaftar
  • cnt: jumlah total sepeda sewaan termasuk yang kasual dan terdaftar

Membuat Visual (Dashboard)

Untuk dashboard, saya menggunakan modul streamlit yang terdapat didalam Python untuk membuat beberapa bahan/chart.

Untuk menampilkan tingkat penyewaan sepeda pada setiap musim, saya menggunakan column char.

Sedangkan untuk Konstribusi penyewaan sepeda per jam, saya menggunakan line chart.

Dapat dilihat pada dashboard bahwa musim gugur memiliki penyewaan sepeda terbanyak dengan rata-rata sewa 5644 di ikuti oleh musim panas dengan rata-rata 4992 penyewa, lalu musim dingin dengan rata-rata 4728 penyewa dan terakhir musim semi dengan rata-rata 2604.

Import Library


import streamlit as st
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

Load data


# Load data
day_df = pd.read_csv('day.csv')
hour_df = pd.read_csv('hour.csv')

Converting datetime


# Convert datetime columns to datetime type
datetime_columns = ["dteday"]

for column in datetime_columns:
    day_df[column] = pd.to_datetime(day_df[column])
    hour_df[column] = pd.to_datetime(hour_df[column])

Mapping season


# Mapping season, yr (year), and mnth (month) columns
season_mapping = {1: "spring", 2: "summer", 3: "fall", 4: "winter"}
yr_mapping = {0: 2011, 1: 2012}
mnth_mapping = {
    1: 'January', 2: 'February', 3: 'March', 4: 'April', 5: 'May', 6: 'June',
    7: 'July', 8: 'August', 9: 'September', 10: 'October', 11: 'November', 12: 'December'
}

day_df['season'] = day_df['season'].replace(season_mapping)
day_df['yr'] = day_df['yr'].replace(yr_mapping)
day_df['mnth'] = day_df['mnth'].replace(mnth_mapping)

hour_df['season'] = hour_df['season'].replace(season_mapping)
hour_df['yr'] = hour_df['yr'].replace(yr_mapping)
hour_df['mnth'] = hour_df['mnth'].replace(mnth_mapping)

clean_day_df = day_df
clean_hour_df = hour_df

rents by season


# Calculate rents by season
rents_by_season = clean_day_df.groupby('season')[['casual', 'registered', 'cnt']].mean().reset_index()
rents_by_season = rents_by_season.rename(columns={'cnt': 'total_average'}).astype({
    'casual': int,
    'registered': int,
    'total_average': int
}).sort_values(by='total_average', ascending=False)

contribution by hour


# Calculate contribution by hour
contribution_by_hour = clean_hour_df.groupby('hr').agg(
    avg_casual=('casual', 'mean'),
    avg_register=('registered', 'mean'),
    total_average=('cnt', 'mean')
).reset_index().astype({
    'avg_casual': int,
    'avg_register': int,
    'total_average': int
})

Streamlit Apps


# Streamlit App
st.title("Dashboard Sewa Sepeda")

Display rents by season chart


# Display rents by season chart with different colors and values
st.subheader("Sewa per Musim")
fig_season = plt.figure()
# palette = sns.color_palette("pastel")
bar_plot = sns.barplot(x='season', y='total_average', data=rents_by_season, palette='pastel', hue = 'season')
for index, value in enumerate(rents_by_season['total_average']):
    bar_plot.text(index, value, str(value), ha='center', va='bottom')
st.pyplot(fig_season)

Display rents by season table


# Display rents by season table
st.subheader("Tabel Sewa per Musim")
st.table(rents_by_season.set_index('season'))

Display contribution by hour chart


# Display contribution by hour chart with all values and increased size
st.subheader("Kontribusi per Jam")
fig_hour = plt.figure(figsize=(12, 6))  # Set the figsize parameter to adjust the size
sns.lineplot(x='hr', y='avg_casual', data=contribution_by_hour, label='Casual', marker='o')
sns.lineplot(x='hr', y='avg_register', data=contribution_by_hour, label='Registered', marker='o')
sns.lineplot(x='hr', y='total_average', data=contribution_by_hour, label='Total Average', marker='o')
plt.xticks(contribution_by_hour['hr'])
plt.legend()
st.pyplot(fig_hour)
